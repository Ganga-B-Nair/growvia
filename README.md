# Growvia - Personalized Career and Education Advisor App

## Overview
Growvia is a personalized career and education advisor app designed to help users navigate their educational and career paths. The app incorporates an AI chatbot, aptitude quizzes, career path visualizers, college directories, and timeline trackers, providing a comprehensive tool for users to make informed decisions about their futures.

## Features
- **AI Chatbot**: Interacts with users to provide personalized career advice based on their inputs and preferences.
- **Aptitude Quizzes**: Engages users with a series of questions to assess their aptitudes and interests, offering results that guide their career choices.
- **Career Path Visualizer**: Visualizes various career paths based on user input, displaying potential career options and necessary steps to achieve them.
- **College Directory**: A searchable directory of colleges and educational institutions, including details about courses, admission processes, and more.
- **Timeline Tracker**: Allows users to track their educational and career milestones over time, providing a visual representation of their progress.

## Technology Stack
- **Frontend**: React
- **Backend**: AI integration for chatbot functionality
- **Styling**: Glassmorphism design with a custom theme

## Color Scheme
- Background Color: `#2A2A2A`
- Text Color: `#ECECEC`
- Accent Color: `#4682B4`

## Installation
1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd Growvia
   ```
3. Install dependencies:
   ```
   npm install
   ```

## Usage
To start the application, run:
```
npm start
```
This will launch the app in your default web browser.
